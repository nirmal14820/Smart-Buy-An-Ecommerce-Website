# Smart-Buy-An-Ecommerce-Website
Smart Buy is a PHP-MySQL based e-commerce website for Shivani Electronics. Users can browse products, manage carts, place orders, and track them. Admins can manage users, products, and order status through a secure dashboard.

# Installation and Setup Guide

## Prerequisites

Before running the project, make sure you have the following installed:

* XAMPP (Apache & MySQL)
* Web Browser (Chrome, Firefox, Edge, etc.)

## Step 1: Install XAMPP

Download and install XAMPP from the official website:

https://www.apachefriends.org/

## Step 2: Start Apache and MySQL

1. Open the **XAMPP Control Panel**.
2. Start the following services:

   * Apache
   * MySQL

## Step 3: Copy the Project

Copy the project folder (`ecom`) into:

```text
C:\xampp\htdocs\
```

The project path should look like:

```text
C:\xampp\htdocs\ecom
```

## Step 4: Import the Database

1. Open your browser and visit:

```text
http://localhost/phpmyadmin
```

2. Create a new database named:

```text
ecom
```

3. Click on the database and select the **Import** tab.

4. Choose the provided SQL file:

```text
ecom.sql
```

5. Click **Go** to import the database.

## Running the Project

### Home Page

Open the following URL in your browser:

```text
http://localhost/ecom/index.php
```

### Admin Login Page

Open the following URL:

```text
http://localhost/ecom/admin_area/login.php
```
